# PhoneCode Improvements Patch

This zip contains a draft implementation of the onboarding roadmap:

- `scripts/install.sh`: interactive installer with Recommended, Minimal, Custom, Repair, Dry run, and Uninstall modes.
- `scripts/setup.sh`: compatibility wrapper that forwards to `scripts/install.sh`.
- `phonecode` command is generated by the installer inside Ubuntu.
- `pc` shortcut is generated by the installer.
- `ocode` safe OpenCode wrapper is generated by the installer.
- Optional `oa` shortcut is generated when selected.
- CI workflow for ShellCheck/dry-run smoke checks.
- Updated docs for installer, quick reference, OpenCode safety, troubleshooting, and advanced setup.
- Security, contributing, changelog, PR template, smoke tests, and examples.

## How to apply manually

Copy these files into the repo root, replacing existing files with the same paths.

Then run:

```sh
sh tests/smoke.sh
sh scripts/install.sh --dry-run --profile minimal --yes
```

## Important note

This is an implementation draft, not a tested release on Android hardware. Test the dry-run first, then run on a disposable Termux/proot install before using it as the main installer.
